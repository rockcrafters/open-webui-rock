# Security policy

## Reporting a vulnerability

To report a security issue, you can either:
 - file a Private Security Report at https://github.com/canonical/open-webui-rock/security/advisories/new, or
 - email security@ubuntu.com.

When reporting a security issue, please provide a description of the issue,
the steps you took to create the issue, affected versions, and, if known,
mitigations for the issue.
